A simple overlay that shows CPS

**Requirements**

* LeviLauncher (Android) — <https://github.com/LiteLDev/LeviLaunchroid>
* DisplayCPS — [Releases](https://github.com/RadiantByte/LeviLauncher-DisplayCPS/releases)

**Installation**

* Install LeviLauncher
* Add DisplayCPS mod in LeviLauncher
* Launch MC through LeviLauncher
